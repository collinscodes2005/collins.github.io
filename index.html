To run .NET frameworks may be quite tidious on a Mac os, this is a detailed process on how to easily run "console" VB.net on your vscode

Configuring VB.Net on MACOS

Install Mono from the website

We shall be using the stable version

Mono

Install Mono: You can download the latest version of Mono from the official website and install it on your Mac.

Verify the installation: After installing Mono, open the Terminal and run the command "mono --version" to verify the installation.

Now : install the .NET CLI extension on your VScode

Now : create your first vbnet file, with the file-extension - ".vb"

After saving your file, open the terminal on your vs-code and compile you .vb file with the code 

"vbnc "filename".vb

This generates a ".exe" file with the same file name as your vb file.

To run your vbcode : type in the terminal 
"mono 'filename'.exe"

Your code should run now in the console.
